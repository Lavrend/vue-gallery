# Vue-catalog

## Установить зависимости
```
yarn install
```

### Собрать svg иконки
```
yarn run svg
```

### Собрать локально, запустить дев сервер
```
yarn run dev
```

### Собрать и минифицировать для продакшена
```
yarn run build
```

### Запустить eslint
```
yarn run lint
```

### Запустить юнит-тесты
```
yarn run test:unit
```
